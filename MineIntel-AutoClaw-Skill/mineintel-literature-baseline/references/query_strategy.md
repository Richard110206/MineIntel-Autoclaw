# 检索策略

## 中文关键词

围绕矿业场景检索时，优先组合：

- 技术方向 + 矿井
- 技术方向 + 井下
- 技术方向 + 煤矿
- 技术方向 + 安全监测
- 技术方向 + 工矿自动化
- 技术方向 + 煤炭学报
- 技术方向 + 煤炭科学技术

示例：

```text
计算机视觉 矿井 安全监测
目标检测 煤矿 皮带异物
深度学习 井下 人员定位
机器视觉 煤炭学报 2024
```

## 英文关键词

将技术方向翻译为英文后组合：

- technique + underground mining
- technique + coal mine
- technique + mine safety monitoring
- technique + anomaly detection mining
- technique + object detection mining

示例：

```text
computer vision underground mining safety monitoring
object detection coal mine conveyor belt
deep learning mine safety monitoring
robot inspection underground mine
```

## 论文分层

报告中将论文线索分为两类：

- 经典代表性：约 2020-2023 年，关注方法基础和矿业落地案例
- 最新前沿：当前年份和前两年，关注新模型、新数据、新系统

如果无法确认题名、作者或年份，不要写成确定论文，写成“检索线索”并提示需核验。
