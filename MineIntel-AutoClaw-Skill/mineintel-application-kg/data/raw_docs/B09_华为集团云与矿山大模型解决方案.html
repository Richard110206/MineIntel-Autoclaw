集团云与矿山大模型

大模型的落地路径分为两类：路径一是基于两级云架构的人工智能训练中心加边缘推理方案，1个矿山大模型，覆盖矿山多个业务场景，覆盖采掘机运通等煤矿生产九大领域；路径二是大模型调优舱，面向预算有限，且不具备自主开发能力的客户，提供轻量化大模型落地方案，实现开箱即用。

方案价值

助力井下少人化

• AI智能设备巡检，减少井下巡检人员
• AI智能检测，识别不规范行为，减少人员下井查看
• AI钻孔深度识别，减少审核人员

提升本质安全

• 从人工观察转向智能自动检测，提升安全监管能力
• 提升人员安全意识，规范人员作业行为，减少安全管理对抗
• 从源头减少并消除生产过程中各类风险隐患，提升本质安全

降本增效

• 安监效率有效提升：由40%提升至80%
• 智能煤流调速，单矿可节约电费10万元/年
• 重介分选密度智能控制，提升精煤产率0.2%
